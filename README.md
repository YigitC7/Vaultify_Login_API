# Vaultify Login API

## English

### What is the Purpose of This Project?
This project solves a recurring pain point for backend developers: building a brand-new authentication and login system for every single project. 

With **Vaultify Login API**, developers can dynamically create separate database environments for each of their projects and manage them effortlessly. You are not confined to a single database instance; you can spinning up as many isolated databases as you need, name them accordingly, and run them securely in a completely multi-tenant, isolated fashion.

## Türkçe

### Bu Projenin Amacı Ne?
Bu proje, backend geliştiricilerinin her yeni projede karşılaştığı ve tekrar eden bir sorunu kökten çözüyor: **Her proje için sıfırdan bir giriş (Login) ve yetkilendirme sistemi yazmak.**

**Vaultify Login API** sayesinde geliştiriciler, projeleri için dinamik olarak ayrı veritabanları oluşturabilir ve bunları oldukça kolay bir şekilde yönetebilir. Bu sistemle tek bir veritabanına bağımlı kalmazsınız; dilediğiniz kadar veritabanı oluşturabilir, bunları isimlendirerek güvenli ve birbirinden izole (multi-tenant) bir şekilde kullanabilirsiniz.
